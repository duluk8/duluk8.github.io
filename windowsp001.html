Domain erstellen und Devices hinzufügen mit PowerShell Windows-Server

Schritt 1 — ADDS Services ausgeben

Mit dem Cmdlet "Get-WindowsFeature" werden Informationen zu Features abgerufen, die sowohl zur Installation verfügbar als auch bereits auf einem Computer mit Windows Server oder einer virtuellen Offline-Festplatte (VHD) mit Windows Server installiert sind.

Get-WindowsFeature -Name *ADDS*

Schritt 2 - Installieren von ADDS

Das Cmdlet "Install-WindowsFeature" installiert die angegebenen Features auf einem Computer, auf dem Windows Server ausgeführt wird, oder auf einer virtuellen Offline-Festplatte (VHD), auf der Windows Server installiert ist. Dieses Cmdlet funktioniert ähnlich wie die Installation von Rollen und Features in Server Manager, mit einer wichtigen Ausnahme: Das Cmdlet installiert standardmäßig keine Verwaltungstools für die Features. Um Verwaltungstools wie Snap-Ins auf einem Zielserver zu installieren, müssen Sie den Parameter IncludeManagementTools zu Ihrem Befehl hinzufügen.

Install-WindowsFeature -name AD-Domain-Services -IncludeManagementTools

Schritt 3 - Eigene Domain erstellen

Mit dem Cmdlet "Install-ADDSForest" wird eine Active Directory-Gesamtkonfiguration installiert. "-InstallDNS gibt" an, dass mit diesem Cmdlet der DNS-Serverdienst für den neuen Forest installiert und konfiguriert wird. "-NoRebootOnCompletion" gibt an, dass das Cmdlet den Computer nach Abschluss des Befehls nicht neu startet. Wenn Sie diesen Parameter weglassen, wird der Computer nach Abschluss des Befehls neu gestartet, unabhängig von Erfolg oder Misserfolg. Diesen Cmdlet kannst du auch weglassen (im nächsten Schritt starten wir den Server neu)

Install-ADDSForest -DomainName "meineDomain.com" -InstallDns:$ture -NoRebootOnCompletion:$true

Schritt 4 - Server neu starten

Mit "Restart-Computer" wird der Server neu gestartet

Restart-Computer

Schritt 5 - Lokaler Computer zur Domain hinzufügen

Das Cmdlet "Add-Computer" fügt den lokalen Computer oder Remotecomputer zu einer Domäne oder Arbeitsgruppe hinzu oder verschiebt sie von einer Domäne in eine andere. Es erstellt auch ein Domänenkonto, wenn der Computer der Domäne ohne ein Konto hinzugefügt wird.

Lokal:
Add-Computer -DomainName meineDomain.com -Restart
Remote:
Add-Computer -ComputerName SVName01 -DomainName meineDomain.com -Restart
