Installiere Apache Web Server auf einem Raspberry Pi

Voraussetzungen

Raspberry Pi 
SD Karte
LAN oder Wireless Verbindung
Stromversorgung  
Mindestens einen Bildschrim, Tastatur und eine Maus
Laptop/PC der SD-Karten lesen kann

Schritt 1: OS installieren

Installiere den Raspberry Pi Imager führe die imager_1.7.2.exe aus.

Öffne den Imager und klicke auf "OS auswählen" (choose OS) > Raspberry Pi OS (other).
Wir werden das Raspbarry Pi OS Lite(64bit) nehmen, wenn du kein Raspberry Pi 3/4/400 hast,
dann nimm das Raspberry Pi OS Lite(32bit). Jetzt haben wir das OS ausgewählt.
Als nächstes wählen wir den Speicher (SD-Karte) aus, danach auf "write" klicken.
Je nach Computer kann dies einige Minuten in anspruch nehmen.

Nun haben wir das Raspberry Pi OS erfolgreich installiert.
Nimm dein Raspi und stecke die SD-Karte (oder dein Speicher) mit dem OS ein. Stelle sicher,
dass dein Raspi mit dem Blidschrim und der Maus verbunden ist.
Nun kannst du dein Raspi mit dem Strom verbinden.
Der Raspi bootet und nach ein paar Sekunden, haben wir das OS mit der Command Line auf unserem Raspi.

Schritt 2: Aufsetzen des Apache Web Servers

Nun werden wir den Apache Web Server auf unserem Raspi installieren.
Zunächst stellen wir sicher mit den folgenden Befehlen, dass wir auf dem neusten Stand sind:

sudo apt-get update
sudo apt-get upgrade

Nun sind wir auf dem neusten Stand und können mit dem nächsten Command weitermachen.

sudo apt install apache2 -y

Mit diesem Command installieren wir den Apache Web Server auf unserem Raspberry Pi.
Nice! Du hast dein eigenen Web Server, nun kannst du deine eigene Website hosten. 
Um zu schauen ob der Web Server läuft, können wir unsere IP-Adresse im Browser eingeben.
Brauche diesen Command, um deine IP-Adresse herauszufinden:

hostname -I

Es wird dir deine private IP ausgeben z.B. 192.168.1.50

Jetzt solltest du die default Website von Apache sehen:

Schritt 3: Wie die Website editieren

Um Änderungen vorzunehmen können wir diesen Command benutzen:

nano /var/www/html/index.html

Der Befehl öffnet mit dem Nano Texteditor das index.html Filel in dem Ordner/Directory /var/www/html/



Das war auch schon die ganze Hexerei
Du hostest deine eingene Website auf deinem Raspberry Pi.
