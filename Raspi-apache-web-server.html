Install Apache Web Server on a Raspberry Pi

Requirements

Raspberry Pi 
SD Card
LAN or Wireless connection
Power supply  
At least one screen, keyboard and a mouse
A device that can read SD cards

Step 1: Installing OS

Install the Raspberry Pi Imager lunch the imager_1.7.2.exe an click install.

Open the imager then click choose OS > Raspberry Pi OS (other). We will go
with a Raspbarry Pi OS Lite(64bit) if you dont't have a Raspberry 3/4/400 then go with
a Raspberry Pi OS Lite(32bit).
So now we have choosen the OS. Next we will choose our Storage (SD card) after that we click write.
This might take a few minutes depend on your computer.

So now you have sucsessfully installed the raspberry Pi OS.
Take your raspi and plug the SD Car (or your other Storage) with the OS in. Make sure the raspi
is connected to your screen and keyboard. So now you can connect the raspi with the power.
It will boot and after a few seconds you have the OS with the command line on your raspi.

Step 2: Setup the apache web server

Now we will install Apache Web Server on the raspi. First we make sure we are up to date with these tow command:

sudo apt-get update
sudo apt-get upgrade

When it is done, we are up to date and can move on with the next command.

sudo apt install apache2 -y

With this command you will install Apache Web Server on your Raspberry Pi. Nice! You have your own Web Server,
so you can now host your own website.  â€‹To check if the web server is running, you can enter your IP in your browser.
Enter this command in your command line:

hostname -I

It shows your IP Address e.g. 192.168.1.50

Step 3: How to edit the web page

To make changes to the default web page you can use this cammand:

nano /var/www/html/index.html

The command open the nano text editor to edit the index.html file in the directory /var/www/html/



That's it for this short tutorial.
You now have an apache web server on your raspberry pi and you can host your own Website.
