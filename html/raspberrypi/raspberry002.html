How to install Pi-hole on Raspberry Pi?

February 01, 2023
Python Python Docker

What are the requirements for installing a Pi-hole?

You need a Raspberry Pi with a SD card and a charger. A device which can read SD Cards. Internet connection preferably LAN, but WLAN does it also.

What is a Pi-hole?

Pi-hole is a network-level ad blocking and internet tracker Linux application that acts as a DNS sinkhole and optionally a DHCP server, intended for use in private networks. It is designed for low-power embedded devices with networking capabilities like the Raspberry Pi, but can be installed on almost any Linux machine.
Pi-hole has the ability to block ads in non-traditional locations such as ads on smart TVs and mobile operating systems as well as ads on traditional websites.

How to install Raspberry Pi OS on your Raspberry Pi?

First, we need a compatible operating system (OS) on our Raspberry Pi. I recommend the Raspberry Pi OS Lite 32-bit for a Pi-hole. So go to the device which can read SD cards and open the Raspberry Pi Imager or download it here. Click on choose OS and then under Raspberry Pi OS (other) and choose Raspberry Pi Os Lite (32-bit). Yeah! Right, the lite version without a desktop (GUI), but that's absolutely fine for a Pi-hole. Now, plug in your SD card and under Storage choose your SD Card. As the second last click on the gear/settings icon. So now enable SSH and set a password to login in a futher step with ssh and then hit write.

When the writing process finished, plug out your SD card and put it in the Raspberry Pi. Make sure that your Raspberry Pi has power, your keyboard and screen is connected. So now wait 5 minutes and give the os time to install...
