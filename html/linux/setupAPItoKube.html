Enviornment

For the whole project I will use a Linux-VM and I recommend you to use also a Linux. At least with a VM (Virtual Machine). Setup Linux VM
As code editor we will use Visual Studio Code, we will program with python, for creating our image and container we will use docker and finally we use kubernetes to host our Note-Service.

Installing Visual Studio Code

More about VS Code

How to install VS Code? It's simple. To install it we simply run two commands. Firstly you should make sure, that your system is up to date. Click on the "Show Applications" and then look for a App named Terminal.

Add it to your favorites by right click and then add.
Now open the Terminal and type in following command:

sudo apt-get update

In this tutorial we will use snap to install VS Code. Snap packages are containerized applications.

sudo snap install --classic code

Verifiy the installation by follwoing command (It should display a version like 1.80.1 or newer):

code --version

To open VS code use code .
If you want you can add it to your favorites by right clicking at the icon at the left and select "add to favorite"

How to install Python?

More about Python

To install python open your termianl once again, and type in following command to check if python is already installed. I would recommend to install python3

python3 --version

Make sure you have the 3.10.6 version or newer. To install python3 use following command:

sudo apt-get install python3

If you want you can check the version again you can do it by typing python3 --version. So congrats you have now successfully installed python. Let's move on with the next one.

How to install Docker on a Linux machine?

More about Docker

First, let's open the termianl which should be displayed at the left or it is already open from the last steps. Then we use this command to install Docker (by the question: Y and enter to continue):

sudo apt install docker.io

To check if the installation of Docker was successfull type docker in the terminal. Now you should see many options for using the docker command.

How to install Kubernetes?

More about Kubernetes

To install Kubernetes we use the kubectl binary with curl. With this command we install the latest release of kubectl:
curl -LO "https://dl.k8s.io/release/$(curl -L -s https://dl.k8s.io/release/stable.txt)/bin/linux/amd64/kubectl"
If the curl command not found message appears, you can install curl with following command:
sudo apt install curl
and then try it again.

Now we can validate the binary with the kubectl checksum. To do that first we need to install the kubectl checksum:
curl -LO "https://dl.k8s.io/$(curl -L -s https://dl.k8s.io/release/stable.txt)/bin/linux/amd64/kubectl.sha256"
Now we can vaildate the kubectl binary against the checksum file:
echo "$(cat kubectl.sha256) kubectl" | sha256sum --check
Now you should see a output like this: kubectl: OK
If the check failed then you should see: kubectl: FAILED
Important: Download the same version of the binary and checksum.
Let's start with the download of kubectl:
sudo install -o root -g root -m 0755 kubectl /usr/local/bin/kubectl
Last but not least, we checkt the version of kubectl:
kubectl version --client
